**Проект: Анализ статистики ДТП и влияния погодных условий на ДТП**

**Стек основных библиотек python:** sqlalchemy, requests, json, BeautifulSoup, pandas, matplotlib, numpy, scipy

**Цель проекта:** посчитать и проанализировать статистические показатели ДТП за период 2015-2025 гг. Провести исследовательский анализ данных: оценить влияние погодных условий на количество ДТП, выявить иные факторы и аномалии при наличии. Создать аналитическую панель (дашборд) с основными показателями. Анализ проводился на базе сравнения двух городов: г. Екатеринбург и г. Владивосток. Выбор городов обусловлен разностью их климата. В Екатеринбурге климат умеренно-континентальный с резкими колебаниями температур и выраженными сезонами. Во Владивостоке климат умеренный муссонный с ярко выраженными сезонами и  высокой влажностью из-за близости океана. 

**План работ:**
1. Создание ETL скрипта для сбора данных, работа с API:
  - получение списка городов РФ: парсинг веб-страницы Wikipedia
  - получение координат городов по их названию через геокодер Яндекса
  - получение данных о погоде: через API сайта open-meteo.com
  - получение данных о ДТП через API ГИБДД РФ (stat.gibdd) для выбранных городов
2. Очистка и предобработка данных
3. Организация хранения данных (база данных Supabase) 
4. Исследовательский анализ данных и проверка гипотез:
  - анализ основной статистики по ДТП, в т.ч. в динамике, построение графиков
  - выявление взаимосвязи  ДТП с погодными условиями
  - категоризация данных и расчет корреляции
  - проверка гипотез 
5. Создание аналитической панели на основе полученных данных (дашборд DataLens)

**Источники данных:** 
1.	Wikipedia  - данные со страницы "Список городов России" 
2.	Яндекс.Геокодер - координаты городов
3.	Open-Meteo API – ежедневные данные о погоде  Free Open-Source Weather API | Open-Meteo.com 
4.	stat.gibdd.ru - коды ОКАТО для регионов 
5.	ГИБДД API  - данные о ДТП  stat.gibdd.ru

**Хранилище (Supabase PostgreSQL):**
Основные таблицы:
1.	wiki_cities  - сырые данные из Wikipedia
2.	clean_cities – очищенные данные из wiki_cities
3.	cities_coords – clean_cities, дополненные координатами городов 
4.	екатеринбург_weather – сырые ежедневные данные о погоде Екатеринбурга за 2015-2025
5.	владивосток_weather - сырые ежедневные данные о погоде Владивостока  за 2015-2025
6.	meteo_Vlad_clean –очищенные данные о погоде Владивостока
7.	meteo_Ekb_clean - очищенные данные о погоде Екатеринбурга
8.	OKATO -  служебная таблица: справочник кодов ОКАТО городов РФ 
9.	main_df – основные данные о ДТП
10.	info_dtp_df – данные о дорожных условиях
11.	ts_df – данные о транспортных средствах 
12.	uch_data_df – данные об участниках ДТП
13.	main_df_clean, info_dtp_clean, ts_clean, uch_clean – очищенные данные о ДТП, дорожных условиях, транспортных средствах и участниках ДТП
14.	info_city – служебная таблица: справочник городов с населением 
15.	calendar – служебная таблица: календарь 2015-2025 с датами ДТП
