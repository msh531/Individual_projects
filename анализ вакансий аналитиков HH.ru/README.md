## Анализ вакансий аналитиков HH.ru

**Стек: python (pandas, matplotlib, seaborn, numpy)**

**Цель проекта:** 
выявить различия в предлагаемых вакансиях для аналитиков данных и системных аналитиков, выявить ключевые навыки, требуемые работодателями, описать типовые параметры работодателей (ТОП-работодателей, зарплата, тип занятости, график работы, в т.ч. в разбивке по грейдам), чтобы помочь потенциальным соискателям сориентироваться на рынке труда.

Описание данных: 
данные о вакансиях за первое полугодие 2024 г.

План исследования:

1. выполнить предобработку данных 

2. провести исследовательский анализ данных:

- выявить требуемые ключевые навыки кандидатов
- проанализировать вакансии в разбивке по типу занятости и формату работы, региону
- определить ТОП работодателей по количеству вакансий
- определить средние/медианные зарплаты в разбивке по грейдам
- сформировать типовые портреты соискателей 

**Вывод:** 

• Различия в профессиях: аналитикам данных чаще требуются SQL и Python, системным аналитикам:  BPMN, SQL, Confluence. Это помогает кандидатам точнее планировать обучение.
•  Soft skills идентичны для обеих профессий: коммуникабельный, с аналитическим мышлением, умеющий работать с большим объемом информации и документацией. 
•  Опыт работы: у системных аналитиков чаще требуется опыт 3+ лет, у аналитиков данных 1-3 года
• География: Москва с существенным отрывом лидирует по числу размещенных вакансий обоих типов (почти 65% всех вакансий), т.о. кандидатам из других городов существенно сложнее найти работу.  Удалённые вакансии частично  улучшают ситуацию, но формат удаленной работы более распространен у системных аналитиков (34% всех вакансий), у аналитиков данных – только 17%. 
• Портрет работодателя: основной спрос формируют банки, IT, маркетплейсы, ритейл, телеком. В целом на банковский сектор приходится примерно 17% размещаемых вакансий. Для финансового сектора часто требуется наличие экономического образования и опыт работы в банке – аргумент для специалистов со смешанным профилем.
• Вилки зарплат для junior дата аналитика: 60-90 тыс. руб., middle –  в среднем 150 тыс. руб. Для junior системного аналитика 70-120 тыс. руб., middle –  в среднем 200 тыс. руб. Результат полезен для переговоров о зарплате.
