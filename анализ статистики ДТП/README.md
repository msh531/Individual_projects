**Проект: Анализ статистики ДТП и влияния погодных условий на ДТП**

Стек основных библиотек python: sqlalchemy, requests, json, BeautifulSoup, pandas, matplotlib, numpy, scipy

Цель проекта: посчитать и проанализировать статистические показатели ДТП за период 2015-2025 гг. Посмотреть и оценить влияние погодных условий на количество ДТП. Выявить иные факторы и аномалии при наличии.  Анализ проводился на базе двух городов: г. Екатеринбург и г. Владивосток.

План работ:

1\. Получение данных, работа с API:

&nbsp;   - получение списка городов РФ: парсинг веб-страницы Wikipedia

&nbsp;   - получение координат городов по их названию через геокодер Яндекса

&nbsp;   - получение данных о погоде: через API сайта open-meteo.com

&nbsp;   - получение данных о ДТП через API ГИБДД РФ (gibdd.ru) для выбранных городов

2\. Парсинг полученных JSON данных в датафреймы pandas. Очистка и предобработка данных, загрузка данных в БД (Supabase)

3\. Исследовательский анализ данных: 

&nbsp;   - анализ основной статистики по ДТП, в т.ч. в динамике, построение графиков

&nbsp;   - выявление взаимосвязи количества ДТП с погодными условиями

&nbsp;   - проверка гипотезы и влиянии количества осадков на количество ДТП

&nbsp;   - расчет корреляции признаков (оценка влияния пострадал человек или нет в зависимости от пристегнутого ремня безопасности и алкоголя)
