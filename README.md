## Pet-проекты
### Стек: python (pandas, matplotlib, seaborn, numpy)

### Анализ вакансий аналитиков HH.ru
Цель проекта:  выявить различия в предлагаемых вакансиях для аналитиков данных и системных аналитиков, выявить ключевые навыки, требуемые работодателями, описать типовые параметры работодателей, чтобы помочь потенциальным соискателям сориентироваться на рынке труда. Представлены данные о вакансиях за первое полугодие 2024 г., полученные из API HH.ru. Описание проекта: выполнена предобработка данных, проведен исследовательский анализ данных (в т.ч. проанализированы вакансии в разбивке по грейду, типу занятости и формату работы, региону и сформированы типовые портреты соискателей)

### Анализ макроэкономических показателей стран мира
Цель проекта: проанализировать ключевые финансовые показатели стран мира за последние 5 лет (2020-2024), выделить лидеров роста и аутсайдеров, оценить динамику показателей по группам стран. Источник данных: данные Международного Валютного Фонда (МВФ) по состоянию на апрель 2025. Описание проекта: проанализирована динамика и объем ВВП по странам и группам стран, динамика численности населения и безработица, инфляция и объем государственного долга. Отдельно проанализированы показатели по России в сравнении с другими странами.
